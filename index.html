点阵LED卖萌表情包(*_*)+假唱机器人

前段时间在创客群里看到一个用点阵LED屏做的超级萌萌哒的小玩偶，有声音LED屏就会有动画，非常有趣，恰好我有一个点阵LED，于是决定自己也做一个，在instructables上找了下发现真的有这个东西
叫Lip Syncing Robot原版是直接把点阵屏焊上去Arduino pro mini的，我买的那个是用MAX7219驱动，看了一下作者的程序试着改成芯片驱动的，改到一半不会了[x_x]，问了一下作者用MAX7219的程序怎么写，第二天作者回复我了，现在任何一款Arduino都可以做了，还留了许多引脚方便以后添加新功能，不多说了，先看下面的图片和视频吧~~~

正面卖萌

侧面Logo

背面开关

效果视频：


有音乐或声音，点阵LED屏的表情就会动，声音越大嘴巴张得越大，没有声音就不开口了卖萌了，哈哈哈
搭配一台MP3小音响在旁边播放的话食用效果更佳[偷笑]，不过后来发现用纽扣电池供电不稳定，大家最好改用小的锂电供电
看完是不是感觉有点萌，想做一只玩，或者想做来送女孩子，女朋友呢，下面就放出怎么做的和一些程序建模文件。

零件：

  • 8X8点阵LED(MAX7219)(32mm x 32mm)
  • Arduino pro mini(其它的Arduino板子也可以)
  • 声音传感器
  • 开关按钮
  • 纽扣电池(最好改用小锂电)
  • 杜邦线(导线)
  • 外壳(需3D打印)



后来又加了两只手上去，感觉更萌。外壳文件已经是加了手的版本
外壳文件外壳是Solidwork2016建模的，听说SW文件低版本打不开
自己到某宝上选购零件吧，Arduino pro mini需要买一个USB转TTL的模块做程序下载器，第一次玩的要买一个
外壳文件自己找3D打印机打印出来

接线图：
电路按照上图接就ok了，当然你也可以修改程序里的引脚，按自己想的去接。声音传感器把原来的排针去掉，不然装不进外壳
DIN—-D2
CS —-D3
SLK—-D4
OUT—-A4
程序文件里面有两个程序，分别是表情包程序和假唱机器人程序。
下载解压后，用Arduino IDE打开程序烧录进去pro mini就可以去卖萌了。
LED-Matrix-Editor这是个在线编辑点阵LED图案的工具，可以自己改成自己喜欢的图案，然后复制代码到表情包程序里修改表情。

一起做来玩吧。

组装穿越机


暑假期间花了差不多1个月的时间组装了一架最近几年才流行起来的竞速无人机(穿越机)，把自己学习和组装无人机的经验写个帖子记录并分享一下，方便一下想组装的新手
想组装首先要对无人机有个深入的了解，然后去淘宝选购心仪的零件，建议把Krellzou的视频教程过一遍，fpvrace这是他的社区网站，有许多新手的视频教程，
无人机理论知识部分可以到魔豆窝这个网站上学习，比较详细。并且加几个穿越机爱好者群进去潜水学习：大力电子的KFP穿越技术讨论群—>313787977；

替代Arduino IDE的Sublime Text的插件---Stino

Stino是Sublime Text的插件，它能把Sublime Text变身成Arduino IDE，但功能比官方软件强大，比如Arduino IDE没有代码自动补全提示。
我第一次用Sublime Text的Stino插件后就爱上了这个玩意，打代码时自动提示和补全，很赞！！并且编译速度比官方IDE快！因为Arduino IDE是基于Java的Processing，Java的虚拟机机制导致编译速度较慢，Sublime Text是C/C++语言编写的，编译速度自然杠杠的！
接下来说说Sublime Text如何安装Stino插件及一些问题的解决方法！
Arduino IDE需要安装到盘的根目录下而不是Program Files目录下！
Stino插件官网
安装需要:

  • Sublime Text <–点击下载
    经测试Sublime Text 2和Sublime Text 3均可使用Stino插件
  • Arduino IDE <–点击下载
    Sublime Text+Stino插件依赖于Arduino IDE，装上哈
